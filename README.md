## Hacknet-Pathfinder是什么？

- 一个为《Hacknet》设计的全面模组开发接口与加载器，可以让Hacknet拥有几乎无限扩展性。  
  [项目地址](https://github.com/Arkhist/Hacknet-Pathfinder)

## 安装Hacknet-Pathfinder。

访问项目Hacknet-Pathfinder的[Releases](https://github.com/Arkhist/Hacknet-Pathfinder/releases)下载安装的工具

### 使用安装程序或py脚本安装

- Windows安装  
   _使用Hacknet-Pathfinder Releases 提供的exe安装程序安装_
  程序会自动读取你游戏的安装路径，如果没有读取请自行填入游戏安装路径。

  然后点击install进行安装。

  如遇到无法安装请更换网络环境，和使用手动安装。
  _使用Hacknet-Pathfinder Releases 提供的py脚本安装_
  双击进行安装（需安装python和tk）
  安装过程与安装程序一致

---

- Linux安装
  _使用Hacknet-Pathfinder Releases 提供的py脚本安装_  
   双击进行安装（需安装python3和tk）
  安装过程与Windows安装一致
  安装完成需为 StartPathfinder.sh文件添加可执行权限

### 手动安装

- Windows安装
  _使用Hacknet-Pathfinder Releases 提供的zip文件安装_
  将压缩包内的所有文件复制到游戏安装目录内替换  
  运行PathfinderPatcher.exe

---

- Linux安装
  _使用Hacknet-Pathfinder Releases 提供的zip文件安装_  
   将文件夹中的 Hacknet.bin.x86_64文件复制并命名为​ HacknetPathfinder.bin.x86_64  
   确保文件名与生成的 HacknetPathfinder.exe对应，以匹配 64 位 Linux 架构。
